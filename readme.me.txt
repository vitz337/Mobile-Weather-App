Weather Forecast Mobile App
This is a React Native App built using Expo. 
This App will allow users to search for current weather conditions anywhere around the world.

Enter city name to view weather data.
Toggle between Celcius and Fahrenheit.
View upcoming weather trends.
User friendly error message IF API fails or city is not found.

Tech stack:
React Native
Expo
Javascript
OpenWeather API

Installation and Running:
1. Clone Repository
2. Install dependencies - npm install
3. Start the expo server - npx expo start
4. Scan QR Code to run the app on your phone.
